事件高级

监听注册事件

eventTarget.addEventListener(type,listener,[useCapture]) 事件监听方式 （IE9）
eventTarget.attachEvent(eventNameWithOn,callback) 事件监听方式 （旧版）

删除事件

eventTarget.onclick = null 传统方式删除事件
eventTarget.removeEventListener(type,listener[,uesCapture]); 监听删除事件

事件对象

e.preventDefault(); 阻止跳转行为
e.target 返回触发事件的对象
e.type 返回类型
e.stopPropagation() 阻止冒泡

事件委托



鼠标事件

contextmenu 控制显示右键菜单
selectstart 禁止鼠标选中
e.clientX 鼠标在x坐标 （可视区域）
e.clientY 鼠标在y坐标 （可视区域）
e.pageX 鼠标在x坐标 （文档区域）
e.pageY 鼠标在y坐标 （文档区域）

跟随鼠标
<style>

<script>
    var div = document.querySelector('div');
    document.addEventListener('mousemove', function(e) {
        var x = e.pageX;
        var y =e.pageY;
        console.log(x,y);
        div.style.top = y - 100 + 'px';
        div.style.left = x - 100 + 'px';
    })
</script>

键盘事件

onkeyup 按键松开
onkeydown 按键按下
onkeypress 按下时触发 (不识别功能键 区分大小写)

    var text = document.querySelector('.jd');
    var con = document.querySelector('.con');
    text.addEventListener('keyup' ,function ( ) {
        con.style.display = 'block';
        con.innerHTML = this.value;
        if(text.value == '') {
            con.style.display = 'none';
        }
    })
    text.addEventListener('blur', function() {
            con.style.display = 'none';
        })

        if(this.value !== '') {
            con.style.display = 'block'
        }
    })
</script>
